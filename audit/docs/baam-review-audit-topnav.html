Design notes
  • One primary nav · everywhere. Same height (64px), same brand, same right-edge pattern. Only the active link + the right-side cluster (CTA vs user-chip) change.
  • Brand is "BAAM · REVIEW AUDIT" — the word AUDIT in gold to distinguish from the sister product. Tagline italic, hides under 720px.
  • Review platform cross-link is visually de-emphasized — small caps, gray, divider in front, ↗ glyph. Reads as "and there's also our other product over there" without competing with the primary CTA.
  • Result page gets a sub-bar for per-audit chrome (business name, downloads, language toggle, "new audit" shortcut). Keeps the primary nav clean and the audit context always visible at the top.
  • Mobile — at narrow widths the right-side links collapse into a hamburger; the brand stays visible, CTA button stays visible. (Not shown in this prototype; can add when we implement.)
